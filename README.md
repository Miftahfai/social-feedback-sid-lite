# Social Feedback SID-Lite

A PsychoPy-coded EEG-friendly social incentive delay task with social-media-style self and other posts.

## Current design

- Self posts use participant nickname + short typed self-post + chosen avatar
- Other posts use standardized usernames + standardized short text + avatars
- Reward and punishment cue conditions
- Adaptive RT thresholding targeting roughly 60% wins

## Main file

- `social_feedback_sid_lite.py`

## Notes

This task was adapted as a simplified lab-friendly paradigm informed by:

Nicolaou, S., Julià, A., Otero, D. et al. Reward-related neural activation during social media exposure in young women with non-suicidal self-injury: evidence for a continuum of severity in the reward network. *Translational Psychiatry*, 15, 308 (2025). https://doi.org/10.1038/s41398-025-03536-8

This repository is an inspired simplified implementation, not a reproduction of that study protocol.
